# Giphy

This is an app for fetching gifs using the Giphy API.

## How to use the App?

```
1. Clone the project.
2. Cd to the cloned directory.
3. npm install.
4. npm run dev.
6. Use the App by typing a word in the input field and you'll get the gifs displayed.
```
